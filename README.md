Для запуска в локальной сети требуется установка docker

Создаём файл .env
В нём должны лежать 3 переменные: 
TELEGRAM_TOKEN - токен вашего бота в телеграме 
ADMIN_ID - численный id аккаунта, на который будут приходить сообщения
WEBHOOK_URL - адрес, на который будут приходить запросы.
для локального запуска рекомендую использовать ngrok или cloudflared tunnel

Скачиваем все необходимые модули
pip install -r requirements.txt

Запускаем приложение через docker
docker-compose up --build
